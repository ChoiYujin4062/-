# 프로그래머스 삼총사

## 풀이에 대한 설명
1. 주어진 문제의 삼총사를 만들 수 있는 방법의 수를 담기 위한 전역변수answerCount와 세 학생의 점수를 더할 전역변수sum을 선언.
2. DFS() 함수에 number배열과 시작인덱스, 현재 뽑은개수를 담아서 호출
3. DFS() 함수 내에서 start, 즉 3명을 다 뽑았을 때만 합계(sum)를 확인하여 그 값이 0일때만 answerCount를 1증가시키고
4. 반복문으로 DFS함수를 재귀호출 하는데,(현재 위치 부터 배열 끝까지 돌면서 선택)
5. 이때 sum에 현재 탐색중인 number[i] 값을 더하고 재귀호출이 끝나면 다시 원래대로 돌려놓기 위해 sum에 number[i] 값을 빼준다.
6. 이제 answer에 answerCount 값을 받아 return


## 풀어나가는 과정에서 시행착오
맨 처음에는 재귀호출이 아닌 반복문으로 해당 문제를 구현하고, 그 이후에 재귀호출로 구현.
반복문으로 풀땐 굉장히 쉬웠지만 재귀호출로 구현하려니 생각해야 할 점이 많아져서 약간 시간이 걸린 문제.
