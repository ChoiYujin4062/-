# 백준 11004번

## 수 N개 A1, A2, ..., AN이 주어진다. A를 오름차순 정렬했을 때, 앞에서부터 K번째 있는 수를 구하는 프로그램을 작성하시오.

## 풀이에 대한 설명
1. 변수 N과 변수 K를 선언
2. N개의 수를 입력받을 배열 arr을 선언하고 반복문으로 입력받음
3. Arrays.sort() 함수를 사용하여 입력받은 배열의 원소들을 정렬하고
4. K번째에 있는 숫자를 출력

## 풀어나가는 과정에서 시행착오
자바 라이브러리 함수인 Arrays.sort() 함수를 사용하여 매우 간단하게 풀은 문제지만...
이렇게 풀면 공부가 안되지 싶어 직접 내 손으로 정렬을 구현하려 했으나 계속해서 시간초과가 뜨는 바람에 일단 Arrays.sort() 함수를 사용하여 풀은 코드를 commit한 상태.
이후에 시간초과가 뜨지 않는 O(N)정렬을 구현하여 다시 제출해야 할 것.
