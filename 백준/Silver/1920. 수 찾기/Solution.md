# 백준 1920번

## N개의 정수 A[1], A[2], …, A[N]이 주어져 있을 때, 이 안에 X라는 정수가 존재하는지 알아내는 프로그램을 작성

## 풀이에 대한 설명
1. N개의 정수를 입력받기 위한 변수 N을 선언
2. N개의 정수를 저장하기 위한 집합 set1을 선언하고
3. N번 만큼 값을 입력받아 집합 set1에 추가하는 반복문 사용
4. X라는 정수를 입력받는 변수 M을 선언
5. 입력받은 M만큼 반복하는 반복문에서
6. num이라는 변수를 선언하여 매 번 값을 입력받고
7. 이 값이 집합 set1에 포함되어 있다면 "1"을 출력, 아니면 "0"을 출력한다

## 풀어나가는 과정에서 시행 착오
맨 처음엔 단순하게 생각해서 배열 2개를 선언하여 값을 입력받고, 반복문과 if문으로 배열1과 배열2의 원소가 같다면 1을 출력, 아니면 0을 출력하는 방법으로 코드를 작성하려 했으나,
해당 방법으로는 두 배열의 원소가 다를 때 마다 0을 출력하여 해결이 안되는 것을 깨닫고 HashSet 클래스를 이용하여 간단하게 풀이하였다.
