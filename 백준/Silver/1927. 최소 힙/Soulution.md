# 백준 1927번

## 널리 잘 알려진 자료구조 중 최소 힙이 있다. 최소 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성하시오.

## 풀이에 대한 설명
자바의 PriorityQueue 클래스를 이용하여 문제 해결
1. 문제에서 요구하는 조건에 따라 연산의 개수 N을 입력받는 변수 N을 선언
2. PriorityQueue 배열 pq 선언
3. 입력으로 0이 들어올 때, 배열이 비어있다면(isEmpty) 0을 출력하고, 비어있지 않다면 배열의 첫 번째에 있는 원소(가장 작은 값)를 출력하고 제거
4. 그 외에 입력으로 0이 아닌 값이 들어오면 배열에 값을 추가
5. 연산의 개수 N 만큼 반복

입력으로 0이 들어올 때, 반드시 0(배열이 비어있는 경우) 또는 가장 작은 값인 첫 번째에 있는 원소를 출력하므로 0이 들어온 횟수만큼 답을 출력한다.

## 풀어나가는 과정에서 시행착오
PriorityQueue 클래스를 사용하면 항상 최소 힙 구조를 유지하기에 쉽게 풀 수 있었던 문제.
