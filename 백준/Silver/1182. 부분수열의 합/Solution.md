# 백준 1182번

## N개의 정수로 이루어진 수열이 있을 때, 크기가 양수인 부분수열 중에서 그 수열의 원소를 다 더한 값이 S가 되는 경우의 수를 구하는 프로그램을 작성하시오.

## 풀이에 대한 설명
재귀호출을 이용해 부분수열을 찾는 문제를 변형.<br>
주어진 문제를 해결하기 위해 기존의 DFS 함수 내에서 수열의 원소를 모두 더한 값이 S인지 확인하기 위해 S를 추가로 전달받게 변경
1. main 함수 안에서 DFS를 호출할 때 추가로 S또한 입력받는다.
2. 경우의 수를 구하기 위해 전역 변수 count 선언.
2. DFS함수에서 재귀호출이 사용되어 배열의 모든 부분수열을 찾을 수 있는데,
3. 이때 재귀호출이 반복되는 동안 찾은 부분수열의 원소를 모두 더한 값을 sum에 저장하고
4. 현재 selected 배열이 비어있지 않으면서 sum 값이 s와 같은 경우에만 전역 변수인 count의 값을 1 증가시킨다.
5. main 함수에서 count 값을 출력하면 크기가 양수인 부분수열 중에서 그 수열의 원소를 다 더한 값이 S가 되는 경우의 수를 구할 수 있다.

## 풀어나가는 과정에서 시행착오
DFS 함수<br>
if((!selected.isEmpty()) && (sum == S)) count++;<br>
이 부분에서 !selected.isEmpty(), 즉 배열이 비어있지 않은 경우를 제외한다라는 조건을 생각하지 못했다.<br>
수열의 크기가 양수인 부분수열을 모두 찾게 되면 공집합인 [] 도 0으로 인식하기 때문에, 위의 조건이 없다면 S값을 0으로 입력할 때 경우의 수가 +1 되는 오류가 발생한다.
