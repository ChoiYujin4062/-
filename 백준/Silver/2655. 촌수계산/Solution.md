# 백준 2655번

# 여러 사람들에 대한 부모 자식들 간의 관계가 주어졌을 때, 주어진 두 사람의 촌수를 계산하는 프로그램을 작성하시오.

# 풀이에 대한 설명
예제 입력 1을 그대로 표현해본다면
<pre>
        1              4
      /   \          /   \
    2      3        5     6
  / | \
7   8  9
</pre>
이런 같은 형태가 됨<br>
※힌트※두 노드간의 최단 거리를 계산하는 문제 -> 구조는 트리 구조지만, 양방향 탐색이 가능해야 하므로 두 간선을 만들어야 함(양방향 그래프)<br>
문제에 부모 자식 간의 관계를 나타내는 두 번호가 주어짐 -> 행이 m, 열이 2인 2차원 정수 배열(간선 정보)<br>
BFS 탐색 사용

1. 전체 노드의 수, 그래프를 표현할 인접 리스트 배열 선언
2. BFS 구현

mian 함수
1. 전체 사람의 수 N(전체 노드 수), 촌수를 계산해야 하는 서로 다른 두 사람의 번호 num1, num2, 부모 자식 간의 관계인 간선 정보를 저장할 2차원 배열 edge
2. 인접 리스트 배열 초기화
3. 인접 리스트에 주변 노드의 정보를 추가할 때, 양방향 그래프이므로 neighbors[a].add(b)/neighbors[b].add(a)

# 풀어나가는 과정에서 시행착오
문제만 봤을 땐 트리 구조에서 최단 거리를 계산하는 문제라고 생각했으나 힌트를 보고 양방향 그래프 구조라는 것을 확인하고 풀어서 구현할 수 있었던 문제.
